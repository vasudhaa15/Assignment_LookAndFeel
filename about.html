About the Developer

Get to know the creator behind the Interior Studio experience

Hello, I'm Vasudhaa Chowdary Nekkanti

I'm a passionate and dedicated developer with a strong foundation in software engineering, data analysis, and user interface design. Currently pursuing a Masters degree in Artificial Intelligence and Business Analytics from the University of South Florida, I combine technical depth with creativity to deliver thoughtful digital experiences.

My academic and professional background includes experience with full-stack web development, quality assurance testing, and enterprise CRM platforms. This website Interior Studio, was built as a demonstration of my ability to design and implement an elegant, interactive platform with clean structure, responsiveness, and usability at its core.

Development Process

The project began with visual planning in Figma, where layouts and interface components were designed with accessibility and aesthetics in mind. The front-end was developed using semantic HTML and CSS, styled with Bootstrap to ensure responsiveness across screen sizes. Interactive functionality such as the dynamic shopping bag was built using vanilla JavaScript and localStorage to manage user selections.

I followed a modular approach to structure HTML and CSS, ensuring code readability, reusability, and easy maintenance. Elements like the cart, navigation, and product cards were implemented with smooth transitions and minimalistic design principles.

Technologies Used

  • HTML, CSS, JavaScript (Vanilla)
  • Bootstrap for responsive layout and grid
  • Figma for UI design and wireframes
  • Git & GitHub for version control
  • localStorage API for persistent cart functionality
